E-commerce Platform

Application web full-stack avec Php, Python et base de données Sql

À Propos du Projet

Cette plateforme e-commerce est une application web complète qui intègre une base de données sophistiquée de comptes IPTV. Le système permet aux utilisateurs de parcourir, sélectionner et acheter des abonnements IPTV à travers une interface intuitive et sécurisée. La base de données a été spécialement conçue pour gérer des comptes IPTV fictifs à des fins de démonstration, permettant de simuler un environnement réel de vente d'abonnements. L'application inclut un système d'authentification robuste, un panier d'achat dynamique, et un processus de paiement sécurisé. Les utilisateurs peuvent gérer leur profil, consulter leur historique d'achats et accéder à leurs informations d'abonnement depuis leur espace personnel.

Technologies Utilisées

  • Php
  • Python
  • Sql
